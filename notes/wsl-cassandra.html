Cassandra on WSL

Hello, this is article is the first of a series of articles on how to install various softwares and troubleshooting in WSL2. WSL is short for Windows Subsystem for Linux. The main attraction of WSL to me was they shared the same file system which makes it easier to access a file than switching on a virtual machine which consume a lot of space and are slow.
In this article, we will see how to install Cassandra on WSL2.
